# visBEEPmbp

This software runs on any modern web browser and is used to visualise the outputs from BEEPmbp. 
When an analysis is run the file "visBEEPmbp.html" in the output directory can be opened to display the results.

The directory "vis-files" contain the actual javascript files used to run the tool along with "BEEPmbp_output.js", a file generated by BEEPmbp that contains all the output information encoded in JSON format.
